

I want to create a web app for users like the persona in <./design.md> (but not
restricted to them):  they specify what use cases they have. Then their system is
probed by somethig described in <./decide-specs-web-app-prompt.md>. The app
finally suggests a download from <./llamafile.txt> and tell the user what they
need to do next.

The final step is that the user has a chat interface running like ChatGPT by
running a local webserver. 

Keep in mind that it is important for us that the sustainablity aspect is used
when choosing the right llamafile (although this aspect may not be that
important for the user).
